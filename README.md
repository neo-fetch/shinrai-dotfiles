# shinrai-dotfiles

Dotfiles of Shinrai, my laptop. This repo contains my dotfiles and configuration.

## Configs Included

- Alacritty
- Neofetch
- Dunst # Needs more work but it's a good start
- Rofi
- Qtile WM
- ZSH
